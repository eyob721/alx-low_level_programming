# 0x07. C - Even more pointers, arrays and strings

## Mandatory

-   0-memset.c
    -   a function that fills the memory with a constant byte. Similar to the string function memset().
-   1-memcpy.c
    -   a function that copies memory area. Similar to the string function memcpy().
-   2-strchr.c
    -   a function that locates a character in a string. Similar to the string function strchr().
-   3-strspn.c
    -   a function that gets the length of a prefix substring. Similar to the string function strspn().
-   4-strpbrk.c
    -   a function that searches a string for any of a set of bytes. Similar to the string function strpbrk().
-   5-strstr.c
    -   a function that locates a substring. Similar to the string function strstr().
