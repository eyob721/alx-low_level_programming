# 0x07. C - Even more pointers, arrays and strings

## Mandatory

-   0-memset.c
    -   a function that fills the memory with a constant byte. Similar to the function memset().
-   1-memcpy.c
    -   a function that copies memory area. Similar to the function memcpy().
-   2-strchr.c
    -   a function that locates a character in a string. Similar to the string function strchr().
